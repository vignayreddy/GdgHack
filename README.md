# NyxWell — Youth Mental Wellbeing Platform

**NyxWell** is a safe, anonymous space for Indian youth to check in, find support, and connect with peer mentors. Built with a **Zen Brutalism** aesthetic, it focuses on privacy, accessibility, and community impact.

## 🚀 Features

- **Daily Mood Tracker**: Log emotions anonymously and visualize trends.
- **Sleep & Stress Analytics**: Monitor wellness metrics with interactive charts.
- **Community Hub**: Join supportive groups and discuss mental health topics.
- **Peer Mentorship**: Connect with trained volunteers for one-on-one sessions.
- **NGO Dashboard**: A professional portal for organizations to track aggregate impact and manage cases securely.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JS, CSS3 (Modern Zen Brutalism)
- **Backend**: Firebase (Firestore, Auth, Hosting)
- **Analytics**: Chart.js for data visualization
- **Maps**: Leaflet.js for regional impact monitoring

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Configure Firebase**:
   Update `js/firebase.js` with your own Firebase project configuration.
4. **Run locally**:
   ```bash
   npx serve .
   ```

## 🔒 Privacy & Security

- **Anonymous by Default**: No names or PII required for youth accounts.
- **E2E Encrypted Sessions**: Support chats are secured using end-to-end encryption.
- **Siloed NGO Data**: Organizations only see aggregated data, never individual logs.
